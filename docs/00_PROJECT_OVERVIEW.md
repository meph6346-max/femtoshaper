# 00. 프로젝트 개요

## 프로젝트명
FEMTO SHAPER — 독립형 3D프린터 진동 분석기

## 한 줄 요약
ESP32-C3 + ADXL345로 프린터 공진 주파수를 측정하고, 인풋쉐이퍼 최적 파라미터를 자동 추천하는 $3 장치.

## 문제 정의
3D프린터 고속 출력 시 **고스팅**(ringing)이 발생한다. 프린터 기계 구조의 고유 공진 주파수에서 진동이 증폭되기 때문이다. **인풋쉐이퍼**는 이 공진을 상쇄하는 디지털 필터이며, 적용하려면 공진 주파수를 **정확히 측정**해야 한다.

## 기존 솔루션의 한계
| 방법 | 한계 |
|------|------|
| Klipper ShakeTune | Klipper 전용. G-code chirp 필요. Marlin/RRF 불가 |
| Klipper ADXL 내장 | Klipper 전용. SPI 보드 배선 필요 |
| ADXL + Raspberry Pi | $35+. 설정 복잡. 비전문가 진입장벽 |
| 수동 튜닝 | 테스트큐브 반복. 시간 낭비. 부정확 |

## FEMTO의 해결책
- **Zero Coupling:** 프린터 보드에 선 하나 안 꽂음. 센서 부착 + USB-C 전원만
- **펌웨어 무관:** Marlin, Klipper, RepRapFirmware 전부 지원
- **$3 BOM:** ESP32-C3 SuperMini($1.5) + ADXL345($1.5)
- **"붙이고 출력하면 끝":** 캘리브레이션 30초 → 아무 모델 출력 → 결과 확인

## 핵심 혁신: Print Measure
기존은 **제어된 chirp 스윕**으로 측정. FEMTO는 **일반 출력 중 가감속 충격파**를 이용.

프린터가 코너를 돌 때 감속→정지→가속 과정에서 힘이 급변하고, 구조물이 고유주파수에서 자유진동한다. 이것이 ADXL345에 감지되어 FFT→PSD→공진 주파수로 변환된다.

## 대상 사용자
- 3D프린터 사용자 (초보~중급)
- 인풋쉐이퍼를 적용하고 싶지만 Klipper가 아닌 사용자
- 저비용으로 진동 분석을 원하는 메이커
